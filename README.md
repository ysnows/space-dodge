# 🚀 Space Dodge

A browser-based space dodge game built with [Three.js](https://threejs.org/).

**[▶️ Play Now](https://file.enconvo.com/tmp/api/y6YlKOv8MVnjQjiFiDvb.html)**

## Gameplay

- Dodge incoming asteroids before they hit your ship
- Collect gold stars ⭐ for +10 bonus points
- Survive as long as possible — speed increases over time
- You have 3 lives ❤️

## Controls

| Key | Action |
|-----|--------|
| `←` / `A` | Move left |
| `→` / `D` | Move right |
| Touch drag | Mobile support |

## Features

- Procedurally generated rocky asteroids with random shapes
- Particle explosion effects on hit
- Progressive difficulty (speed + spawn rate increase)
- Smooth ship movement with tilt physics
- Animated engine glow
- Starfield background with fog

## Tech Stack

- [Three.js r128](https://threejs.org/) — 3D rendering
- Vanilla HTML/CSS/JS — zero dependencies beyond Three.js
- Runs entirely in the browser, no server needed

## Run Locally

Just open `index.html` in any modern browser — no build step required.

```bash
open index.html
```
